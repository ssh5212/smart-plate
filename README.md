# smart-plate
당뇨 환자들의 식단 조절을 위한 하드웨어(Smart Plate) 및 소프트웨어(서버) 제작입니다.

- 프로젝트 기간 : 2022-07-25 - 2022-10-13

## 개발 기능
- 식판 내 카메라를 통한 음식 촬영
- 식판 내 로드셀을 통한 무게 측정
- 촬영된 음식의 이름을 인공지능 모델을 통해 확인
- DB에 저장된 정보 + 음식명 + 무게를 통한 음식 섭취량 확인
